# Living Dungeon 

Üdvözlünk a **Living Dungeon** világában! Ez egy 2D-s platformer és ügyességi játék, ahol veszélyes csapdákat kell kikerülnöd, éber őrökön kell túljutnod, és meg kell találnod a kiutat a rejtélyes kazamatákból.

##  Főbb Jellemzők
- **Ügyességi platformer:** Ugorj át a szakadékokon, kerüld el a mozgó ellenségeket, és vigyázz a csúszós felületekkel!
- **Kölcsönhatás a világgal:** Beszélgess NPC-kkel hasznos infókért, és használj létrákat a mélyebb szintek felfedezéséhez.
- **Power-upok:** Vedd fel a pályán elszórt gyorsítókat, hogy hatalmasakat tudj ugrani és le tudd hagyni a veszélyt.
- **Kíméletlen világ:** Egy rossz lépés a halálzónába, vagy egy érintés az ellenségtől, és a pálya azonnal újraindul.

## Irányítás
- **Mozgás:** `A` és `D` gombok (vagy Balra / Jobbra nyilak)
- **Ugrás:** `Space`
- **Interakció (NPC beszélgetés, Létra használata):** `E` gomb
- **Célvonal elfogadása:** `Enter`

## Indítás (Setup)
A játék kipróbálásához egyelőre a Unity Editorra van szükség:

1. Töltsd le a projektet a GitHubról (zöld "Code" gomb -> Download ZIP), majd csomagold ki a gépeden.
2. Nyisd meg a letöltött mappát a **Unity Hub** segítségével.
3. A szerkesztőben (alsó Project ablak) navigálj ide: `Assets > The Kazamath > Assets > Scenes`
4. Nyisd meg a tesztpályát (pl. `SampleScene.unity`) dupla kattintással.
5. Nyomd meg a **Play** gombot az ablak tetején, és indulhat a játék!
